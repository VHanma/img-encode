# Gariaev GitHub Safe Import Rules

1. Do not blindly copy code from unknown repositories.
2. Respect licenses.
3. Do not import secrets, keys, tokens, or credentials.
4. Prefer ideas and architecture patterns over direct copy-paste.
5. Best safe imports:
   - frequency registry
   - FFT / Goertzel peak scan
   - RGB histogram imprint
   - reference/object/mirror stereo
   - manifest schema
   - hardware export profiles
6. Keep medical claims out of the app.
7. Keep all human/animal use consent-based, non-invasive, and low-power.
